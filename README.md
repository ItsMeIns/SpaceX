# SpaceX

### Цей проєкт є iOS додатком, який відображає інформацію про ракети SpaceX та їхні запуски.
Подивитись завдання - iOS 2022 Jan.pdf

+ Swift
+ API
+ URLSession
+ JSONDecoder
+ UIKit
+ UITableView
+ UICollectionView
+ PageControll
+ UserDefaults
+ MVVM
